# 授業評価レポート自動送信ツール

Google スプレッドシートの授業評価データをまとめ、各生徒にビジュアルレポートをメール送信するツール。

## 仕組み

```
Google スプレッドシート（評価データ）
    ↓
Google Apps Script が自動実行
    ↓
生徒ごとにグラフ付きHTMLレポートを生成
    ↓
Gmail で各生徒のメールアドレスに送信
```

## スプレッドシートの形式

| 氏名 | メールアドレス | 授業名 | 理解度 | 満足度 | 難易度 | コメント |
|------|--------------|--------|--------|--------|--------|---------|
| 山田太郎 | taro@example.com | 数学 | 4 | 5 | 3 | わかりやすかった |

## セットアップ

1. Google スプレッドシートを作成し、上記の形式でデータを入力
2. スプレッドシートのメニュー「拡張機能」→「Apps Script」を開く
3. `src/main.gs` の内容をコピーして貼り付ける
4. `SPREADSHEET_ID` を自分のシートIDに変更
5. 「実行」ボタンで動作確認
6. トリガーを設定して定期実行を有効化

## ファイル構成

```
src/
├── main.gs       # エントリーポイント・設定
├── sheets.gs     # スプレッドシートからデータ取得
├── report.gs     # HTMLレポート生成
└── email.gs      # メール送信
```
